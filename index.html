Как я использую lazy loading в HTML

Lazy loading помогает не загружать все изображения сразу. Это полезно для длинных страниц, где часть контента находится ниже первого экрана.

Главное изображение

Главное изображение в верхней части страницы должно загружаться сразу, потому что пользователь видит его первым.

Изображение ниже первого экрана

Изображения ниже первого экрана можно загружать отложенно, чтобы не перегружать начальную загрузку страницы.

Ещё один пример lazy loading

На длинных страницах и в галереях lazy loading помогает отложить загрузку менее важных изображений.
